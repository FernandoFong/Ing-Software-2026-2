# Ing-Software-2026-2

Repositorio oficial de ingeniería de software del semestre 2026-2 de la Facultad de Ciencias de la UNAM, donde se subirá el código para las prácticas, así como herramientas para poder configurar un repositorio para desarrollo de software a nivel profesional, con el propósito de que el alumno tenga claro conceptos como:

- Entornos virtuales con Python.
- Conexiones a SQL, manejo de ORM. (PySQLAlchemy)
- Fundamentos de CI/CD.
- Uso de pruebas unitarias. (desarrollo y funcionamiento)
- Creaciones de APIs. (Flask)
- Fundamentos de CSS + HTML. (Jinja)
- Arquitectura de aplicaciones profesionales.

Todo esto para que los alumnos tengan una base para su proyecto final de la materia.

## Organización del repositorio

Dentro del folder de prácticas estarán todas las prácticas que se tendrán durante el curso que incluye el material de código, el PDF de la práctica y el _template_ de LaTeX para contestar las preguntas de cada práctica.
